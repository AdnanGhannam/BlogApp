# Blog App


## Pages:
- Home (All & withTag & searchResults)
- Profile (Home & About & SavedStories)
- Story

## Main Features:
- Login & Signup
- CRUD a story and the profile
- Searching (By 'Author' & 'Title' & 'Tag')
- React to stories
- Save stories
- Share stories
- View users info
- Show simular stories
